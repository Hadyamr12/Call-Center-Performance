# Call Center Performance Dashboard

Interactive Call Center Performance Dashboard built using Power BI and Excel to analyze customer service operations and agent performance.

## Project Overview
This dashboard transforms raw call center data into actionable insights through interactive visualizations and KPI tracking.

## Key Insights
- Call Acceptance Rate
- Call Resolution Rate
- Customer Satisfaction Analysis
- Average Speed of Answers
- Agent Performance Comparison
- Calls Distribution by Topic, Month, and Hour
- Interactive Filtering by Date and Resolution Status

## Tools & Technologies
- Power BI
- Microsoft Excel
- Data Cleaning
- Data Modeling
- Data Visualization

## Dashboard Features
- Interactive slicers and filters
- KPI cards for quick monitoring
- Dynamic charts and performance tracking
- Business-focused analytical insights

## Purpose
The goal of this project is to practice transforming operational data into a professional business intelligence dashboard that supports data-driven decision making.
